# SimpleGauges

* MIT-licence

To add to your Android Project:
* Add in your root build.gradle at the end of repositories:

    allprojects {
    repositories {
     ...
      maven { url 'https://jitpack.io' }
     }
    }

* Add the dependency:

    dependencies {
            implementation 'com.github.hvatum:SimpleGauges:Tag'
    }
