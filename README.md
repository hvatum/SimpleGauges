# SimpleGauges

* MIT-licence

To add to your Android Project:
* Add in your root build.gradle at the end of repositories:

```gradle
allprojects {
 repositories {
  ...
  maven { url 'https://jitpack.io' }
 }
}
```

* Add the dependency:

```gradle
dependencies {
        implementation 'com.github.hvatum:SimpleGauges:0.2'
}
```
